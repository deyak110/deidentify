# PHI De-Identifier Web App

A small Next.js web application that converts the Tkinter PHI de-identification workflow into an accessible browser-based interface.

## Local development

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Deploy to Vercel

1. Push this folder to a new GitHub repository.
2. Sign in to Vercel.
3. Import the GitHub repository.
4. Accept the default Next.js build settings.
5. Deploy.
